可以看出，快速排序和次序选择问题侧重于“分解”这一步的设计，都是以一个基准元素来进行数组划分（提高性能时可采用随机化方法来选择基准元素），“合并”步骤很简单。而归并排序、最大子数组和逆序对计数问题则注重“合并”这一步，采用线性合并的方法，在划分时仅进行简单的两两划分。

动态规划

  • 最优子结构性质

    问题的最优解由相关子问题最优解组合而成

    子问题可以独立求解

    通过最优子结构性质，就可以构造出递推关系，构造出“备忘录表格”，从而解决问题。若想得出最优解对应的最优方案，可根据具体问题设置rec数组来记录，最后根据rec数组来输出最优方案即可。

    子问题可以独立求解

  • 0/1背包问题

    • 蛮力枚举法

    • 带备忘录的递归法（避免了相同子问题的重复计算）

    • 递推法（自底向上来直接构造备忘录表格，从而最终获取最优解）

      状态转移方程：dp(i, c) = max{ dp(i-1, c-vi) + pi , dp(i-1, c) }

      在前i个商品中选择，背包容量为c时的最优解

      即在不选第i个商品和选第i个商品这两种情况中选择最好的那一个结果，**vi**代表商品体积，**pi**代表商品价值

  • 最大子数组问题

    • 状态转移方程：令**D[i]**为以第i个数结尾的最大子数组和，**X[i]**为第i个数的值

      D[i] = max{ X[i] + D[i-1], X[i] }
